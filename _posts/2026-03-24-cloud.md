## What is cloud computing?

From Wikipedia: “Cloud computing is the on-demand availability of computer system resources, especially data storage and computing power, without direct active management by the user. The term is generally used to describe data centers available to many users over the Internet.”

Cloud computing is scalable for the user, an advantage over on-premise computing resources.

## Cloud security

Cloud services at federal agencies need to be secure. [FedRAMP authorized services](https://www.fedramp.gov/) help lower the barrier to adoption by federal agencies.

## Google Workstations at NOAA Fisheries: A cloud computing option

Google workstations are available for NOAA Fisheries staff for cloud computing.

>Google Cloud Workstations is a managed service that provides a flexible, self-serve platform. Because Google handles the underlying infrastructure, the Fisheries Cloud Program team can focus on creating tailored environments, managing access, and ensuring security, reducing setup costs and promoting a faster "time to science" for users.

Request access or learn more in the [Cloud Compute Accelerator Documentation](https://docs.google.com/document/d/1nziPdPULoRWOYQ9WKzISNUgJvANACvfYpFr1z3Ro2Bc/edit?tab=t.0) (must be logged into NOAA email to access document).

Thanks to NOAA Fisheries early adopters who have documented [additional tips and tricks for using Google Workstations](https://github.com/nmfs-opensci/CloudComputingSetup/).

## NOAA High Performance Computing

[NOAA has High Performance Computing resources](https://www.noaa.gov/information-technology/hpcc). NOAA HPC includes both cloud computing and on-prem resource options. This option is worth exploring for workflows with high computational needs.

For NOAA Fisheries staff interested in using NOAA HPC resources, [contact the FIT](https://nmfs-ost.github.io/noaa-fit-resources/contact/) to learn more or submit a request. 

## Cloud Data Hosting and Archiving

### Archiving

NOAA's [NCEI](https://www.ncei.noaa.gov/archive) provides archive services for much of the data from NOAA Observing Systems. Other options include [NOAA approved Data repositories](https://wpo.noaa.gov/wp-content/uploads/2025/08/NOAA-ADRs_DGCFinalDraft_June2025.pdf).

### Cloud Hosting

Generally keeping data “close to computation” will increase the speed and reduce cost of cloud resources. For example, keeping a GCP data bucket in the same project as the VM used for computation is suggested if computation is a big part of the workflow.

NOAA Fisheries staff increasingly have access to GCP data buckets, which have flexible permissions. Put in a ticket with your office's IT support to learn more about what is available at your office.

For public data sets the [NOAA Open Data Dissemination Program](https://www.noaa.gov/information-technology/open-data-dissemination) also provides a pathway for cloud hosting of public data.
